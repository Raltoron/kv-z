Kvízes Oldal

Válassz egy témát, és kezdj el játszani!

Hogyan teheted közzé:

  1. Hozz létre egy GitHub fiókot, ha még nincs.
  2. Készíts egy új repository-t a GitHubon.
  3. Az index.html fájlt töltsd fel a repository-ba.
  4. A repository beállításaiban keresd meg a "Pages" részt.
  5. Állítsd be a "Source"-t a fő ágra (pl. main), majd mentsd el.
  6. Pár perc múlva az oldalad elérhető lesz a megadott linken (pl. https://felhasznalonev.github.io/repository-nev/).
